>  https://github.com/CristianJimenezAI/DevProyects/tree/main/tetris

1. Clona este repositorio en la máquina local.
2. Ejecute el HTML en el buscador de preferencia.
*(Necesita de JavaScript para ejecutar el programa)


Características
----------------------------
- Juego clásico de Tetris con mecánicas básicas
- Controles simples para mover y rotar los bloques, como también acelerar su movimiento
- Puntuación
- Gráficos y diseño con responsive limitados.


Uso
----------------------------
- Utilice las teclas de flecha J y L para mover el bloque.
- Presione la tecla de flecha hacia abajo para acelerar la caída del bloque.
- Pulse las teclas W y Q de flecha hacia arriba para rotar el bloque.
- Intente completar filas para ganar puntos (único nivel de dificultad).
