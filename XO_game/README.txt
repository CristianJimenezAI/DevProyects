	Juego XO
----------------------------
Juego clásico de XO construido con HTLM, CSS y Javascript

----------------------------
No aplica


Instalación
----------------------------
>  https://github.com/CristianJimenezAI/DevProyects/tree/main/tetris

1. Clona este repositorio en la máquina local.
2. Ejecute el HTML en el buscador de preferencia.
*(Necesita de JavaScript para ejecutar el programa)


Características
----------------------------
- Juego clásico de XO con mecánicas básicas
- Uso simple utilizando el mouse para seleccionar el espacio con marca X o O
- Mensaje de alerta en caso de ganar o de reiniciar el juego
- Gráficos y diseño con responsive.


Uso
----------------------------
- Utilice el mouse para seleccionar una de las casillas.
- Cada jugador tiene un turno hasta que alguno gane o ya no hayan mas opciones.
- Al final el juego, debe reiniciar el juego por medio de la ventana de alerta
